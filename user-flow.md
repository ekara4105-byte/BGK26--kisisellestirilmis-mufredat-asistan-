# Flying Focus: User Flow

1. **Dashboard (Ana Sayfa):** - Karşılama: "Bugün enerji seviyen nasıl pilot?"
   - Enerji Seçimi: Düşük, Orta, Yüksek düğmeleri.
   - Durum Özeti: Yaklaşan deadline'lar ve bugün bitmesi gereken görevler.

2. **Müfredat Yönetimi (Curriculum Hub):**
   - Veri Girişi: Ders adı, konu adı, zorluk derecesi ve Deadline (Tarih seçici).
   - Liste Görünümü: Mevcut tüm müfredatın tablolanmış hali (Veritabanından çekilir).
   - Düzenleme/Silme: Konu durumunu (Tamamlandı/Devam Ediyor) güncelleme.

3. **Akıllı Planlayıcı (AI Engine):**
   - "Bugünkü Rotamı Oluştur" butonu.
   - AI, enerji seviyesi + deadline aciliyetini birleştirerek 3 ana görev önerir.

4. **Odak Modu (Focus Session):**
   - Pomodoro veya serbest zamanlayıcı.
   - Tamamlanan görevlerin veritabanında güncellenmesi.
